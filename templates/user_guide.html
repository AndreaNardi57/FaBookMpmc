User Guide

Getting Started
  1. Navigate to the Home page to see existing books
  2. Click "Add Book" in the navigation menu to add a new book
  3. Fill out all required fields in the book addition form
  4. Click "Add Book" to save the new book to the library
Book Management
  • Adding Books: Provide complete information including title, author, ISBN, publication year, and genre
  • Deleting Books: Use the delete button on the home page to remove a book from the library
Troubleshooting

Ensure all required fields are filled and the ISBN is unique.
